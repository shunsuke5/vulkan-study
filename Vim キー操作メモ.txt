Vim キー操作メモ.txt

編集
    x = 一文字削除

    dd = 行削除

カーソル横移動
    w = 次の単語の先頭へジャンプ

    W = 次の空白区切りの先頭へジャンプ

    e = 次の単語の末尾へジャンプ

    E = 次の空白区切りの末尾へジャンプ

    b = 前の単語の先頭へジャンプ

    B = 前の空白区切りの先頭へジャンプ

    ge = 前の単語の末尾へジャンプ

    gE = 前の空白区切りの末尾へジャンプ

    f文字 = 右方向で最初の指定文字にジャンプ

    t文字 = 右方向で最初の指定文字の手前にジャンプ

    F文字 = 左方向で最初の指定文字にジャンプ

    T文字 = 左方向で最初の指定文字の手前にジャンプ

    0 = 行の先頭へジャンプ

    ^ = 空白を除く行の先頭へジャンプ

カーソル縦移動
    - = 空白を除く上の行頭に移動

    H = 画面に表示されている一番上の行へカーソルを移動(High)

    M = 画面に表示されている中央の行へカーソルを移動(Middle)

    L = 画面に表示されている一番下の行へカーソルを移動(Low)

    数字G = 指定した行へ(Go)

    { = 上の空行へ

    } = 下の空行へ

    ]] = カーソルより下にある { へジャンプ

    [[ = カーソルより上にある { へジャンプ

    ][ = カーソルより下にある } へジャンプ

    [] = カーソルより上にある } へジャンプ

    Ctrl + d = 画面半分進む(down)

    Ctrl + u = 画面半分戻る(up)

    Ctrl + y = カーソルを固定して上へスクロール

    Ctrl + e = カーソルを固定して下へスクロール

    zt = カーソルがある行を画面最上部にする(top)

    zz = カーソルがある行を画面中央にする

    zb = カーソルがある行を画面最下部にする(bottom)
