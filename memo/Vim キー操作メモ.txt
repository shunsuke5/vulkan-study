Vim キー操作メモ.txt

インサートモード
	Ctrl + h = BackSpaceと同じ

編集
    s = カーソル下の文字を削除して挿入モードへ

    S = カーソル行を削除して挿入モードへ

    D = カーソルから右側を削除

    C = カーソルから右側を削除して挿入モードへ

    ci" = カーソルがある""の中身を削除して挿入モードへ

    cw = カーソル下の文字から次の単語の手前まで削除して挿入モードへ

    r文字 = カーソル下の文字を指定文字に置換

    x = カーソル下の文字を削除

文字列検索
    /文字列 = 文字列を検索

    n = 次の検索結果へ

    N = 前の検索結果へ

    * = カーソル下にある文字列で検索(単語検索のため、変数名置換などで便利)

    :noh = 検索ハイライトを消す

一括編集
    :%s/置換前文字列/置換後文字列/g

カーソル横移動
    w = 次の単語の先頭へジャンプ

    W = 次の空白区切りの先頭へジャンプ

    e = 次の単語の末尾へジャンプ

    E = 次の空白区切りの末尾へジャンプ

    b = 前の単語の先頭へジャンプ

    B = 前の空白区切りの先頭へジャンプ

    ge = 前の単語の末尾へジャンプ

    gE = 前の空白区切りの末尾へジャンプ

    f文字 = 右方向で最初の指定文字にジャンプ

    t文字 = 右方向で最初の指定文字の手前にジャンプ

    F文字 = 左方向で最初の指定文字にジャンプ

    T文字 = 左方向で最初の指定文字の手前にジャンプ

    0 = 行の先頭へジャンプ

    ^ = 空白を除く行の先頭へジャンプ

カーソル縦移動
    - = 空白を除く上の行頭に移動

    H = 画面に表示されている一番上の行へカーソルを移動(High)

    M = 画面に表示されている中央の行へカーソルを移動(Middle)

    L = 画面に表示されている一番下の行へカーソルを移動(Low)

    数字G = 指定した行へ(Go)

    { = 上の空行へ

    } = 下の空行へ

    ]] = カーソルより下にある { へジャンプ

    [[ = カーソルより上にある { へジャンプ

    ][ = カーソルより下にある } へジャンプ

    [] = カーソルより上にある } へジャンプ

    Ctrl + d = 画面半分進む(down)

    Ctrl + u = 画面半分戻る(up)

カーソルジャンプ
    Ctrl + o = 前の変更箇所へ(older position)

    Ctrl + i = 次の変更箇所へ(incoming position)

画面表示
    Ctrl + y = カーソルを固定して上へスクロール

    Ctrl + e = カーソルを固定して下へスクロール

    zt = カーソルがある行を画面最上部にする(top)

    zz = カーソルがある行を画面中央にする

    zb = カーソルがある行を画面最下部にする(bottom)

繰り返し
    数字コマンド = 指定コマンド指定回数繰り返す

    . = 直前の編集系操作を繰り返す

    ; = 直前の検索系操作を繰り返す
